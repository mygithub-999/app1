Phone Usage Rules

No Phone Usage While Driving Rule

Rule: It is prohibited to use mobile phones while driving within the designated zone.

Fine: Rs. 1000 for the first offense, Rs. 2000 for the second offense, and license suspension for subsequent offenses.

No Phone Usage in Restricted Areas Rule

Rule: Mobile phone usage is prohibited in specific restricted areas marked within the zone, such as libraries, theaters, or classrooms.

Fine: Rs. 500 for the first offense, Rs. 1000 for the second offense, and possible expulsion from the premises for subsequent offenses.

No Phone Usage During Meetings/Events Rule

Rule: Phone usage is prohibited during meetings, seminars, or events within the zone unless explicitly permitted.

Fine: Rs. 250 for the first offense, Rs. 500 for the second offense, and removal from the event for subsequent offenses.

No Phone Usage in Quiet Zones Rule

Rule: Mobile phone usage is prohibited in designated quiet zones within the area, such as meditation rooms or designated study areas.

Fine: Rs. 750 for the first offense, Rs. 1500 for the second offense, and temporary ban from accessing quiet zones for subsequent offenses.

No Phone Usage Near Sensitive Equipment Rule

Rule: Usage of mobile phones is prohibited in areas near sensitive equipment or machinery where electromagnetic interference could pose risks.

Fine: Rs. 200 for the first offense, Rs. 400 for the second offense, and potential liability for any damages caused by interference for subsequent offenses.
